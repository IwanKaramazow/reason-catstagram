# reason-catstagram

## dev tools (Vscode)

- Remove all prior reason plugins
- Download https://github.com/jaredly/reason-language-server/releases
- In the vscode extensions panel, open the menu and click "install from .vsix".

## install

```
yarn
```

## development

```
# run the bsb compiler
yarn bsb
# in another tab
yarn bundle

# open localhost:1234
```
